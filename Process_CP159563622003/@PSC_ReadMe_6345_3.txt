Title: Process/CPU Scheduling Simulation (4 algorithms)
Description: This was my Operating Systems Project for the Spring 2003 semester. It is very interesting and has a nice OOP (object oriented design). It simulates a CPU switching Processes and you can choose between 4 algorithms. this includes:
<br>Preemptive:
<br>i)Round Robin
<br>ii)Shortest Runtime Remaining Next
<br>Non-Preemptive:
<br>FCFS (First Come First Serve)
<br>Shortest Runtime Next
